PENEIRA Dicionário com Filtros

WebService para consulta de palavras do dicionário português brasileiro.

Acessando o WebService

Para acessar o WebService, sem utilizar filtros, você deve acessar o endpoint /api/v1/words.

Filtros das Palavras

A consulta das palavras aleátorias tem diversos filtros de buscas, sendo eles:

?charSize - Retorna uma palavra aleatória contendo os caracteres desejados

"word": "amigo"


?maxChar - Retorna uma palavra aleatória contendo o máximo de caracteres

"word": "cão"


"word": "lã"


?minChar - Retorna uma palavra aleatória contendo o minimo de caracteres

"word": "avião"


"word": "cavaco"

Validação

Alguns filtros podem ser combinados com a utilização do caractere &. Exemplo: ?minChar e ?maxChar podem ser combinados, ficando assim . Dessa forma, o WebService retornará uma palavra entre 2 e 4 letras. Mas tenha cuidado para não fazer consultas com filtros conflitantes!

Quando a consulta é feita com filtros conflitantes, ela é invalidada. Exemplo: ?minChar maior que ?maxChar, ou utilizar ?minChar ou ?maxChar junto com ?charSize. Nesses casos, o código de retorno da consulta será 400 (Bad Request). Antes de acessar o WebService, certifique-se de que os filtros não irão conflitar entre si.
